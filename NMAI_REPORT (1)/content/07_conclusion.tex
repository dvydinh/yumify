\section{Kết luận}
% ===========================================================================

Dự án đã triển khai thành công hệ thống AI tích hợp 5 trụ cột trí tuệ nhân tạo cho bài toán gợi ý ẩm thực đa ràng buộc. Đóng góp chính:

\begin{itemize}
  \item Mô hình hóa toán học đầy đủ 5 trụ cột: Search (A* với Canonical State + Repetition Penalty), CSP (Backtracking + MRV + Dual-constraint Pruning), Logic (Forward Chaining trên Horn Clauses), Bayes (Extended Noisy-OR with Synergy Hidden Causes), ML (Multinomial Naive Bayes + TF-IDF)
  \item Triển khai hoàn toàn bằng Pure Python (chỉ dùng \texttt{numpy}), phù hợp Google Colab
  \item Data pipeline tự động: Kaggle CSV $\to$ stratified sampling $\to$ cost cross-reference $\to$ TF-IDF
  \item 7/7 core modules PASS, notebook chạy Runtime $\to$ Run All không lỗi
\end{itemize}

\subsection{Hướng phát triển}
\begin{itemize}
  \item SentenceTransformers (SBERT) thay TF-IDF cho semantic matching sâu hơn
  \item Tích hợp giá thị trường thực tế thay vì cross-reference heuristic
  \item Reinforcement Learning từ phản hồi người dùng để cập nhật Bayes CPT
  \item Mở rộng NLP hỗ trợ đa ngôn ngữ (tiếng Việt, tiếng Nhật)
  \item Training Naive Bayes trên toàn bộ 230,000 recipes Kaggle (thay vì 5000 sample)
\end{itemize}
